- Current rules:

  - `close >= sma20`
  - `sma20 >= sma60`
- Current score model:
  - `candidate_band = 2`
  - `watch_band = 1`
  - each technical pass = `+1`
  - `max_score = 4`
- Current decision mapping:
  - `candidate`: full market-cap band + both technical confirmations
  - `watch`: market-cap passes watch floor but technical confirmation is partial or missing
